# Public assets

This directory is the only home for static files delivered to MicroKit visitors.

Use `interactions/<interaction-id>/` for component-specific media and `brand/` for shared product assets. Reference files with `/assets/...` paths in the application. Do not put source code, internal notes, or build artifacts here: everything in `public/` is publicly reachable.

`legacy/` contains unused files retained from the original Next.js starter; new work should not add files there.
